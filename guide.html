Your personal preparation companion. Open the app below, then follow the install steps to use it like a native app on your phone or PC.

Open BPSC Tracker →

📱 Install on Android (recommended)

Chrome / Edge / Firefox
Step 1
Open in Chrome on your Android phone
Open the index.html file using a local server or hosted link in Chrome browser
Step 2
Tap the 3-dot menu → "Add to Home Screen"
Chrome will prompt you automatically after a few seconds, or use the menu option
Step 3
Tap "Install" or "Add"
The app appears on your home screen with the 📚 icon. Launch it like any other app — it opens fullscreen without browser chrome

💻 Install on Windows / PC

Chrome / Edge
Step 1
Open index.html in Chrome or Microsoft Edge
Double-click index.html, it opens in your default browser. If needed, right-click → Open with → Chrome or Edge
Step 2
Click the install icon in the address bar
In Chrome: look for the ⊕ icon on the right side of the address bar. In Edge: look for the + icon. Click it.
Step 3
Click "Install"
The app opens in its own window without browser UI. A shortcut is added to your desktop and Start menu automatically

🍎 Install on iPhone / iPad

Safari only
Step 1
Open in Safari (must be Safari, not Chrome on iOS)
Navigate to the index.html file
Step 2
Tap the Share button → "Add to Home Screen"
The share button is the box with an arrow pointing up at the bottom of Safari
Step 3
Tap "Add"
The app icon appears on your home screen
💡 To use on multiple devices: Use the Export button in the app to download a .json backup file, then use Import on the new device to restore all your progress, notes and settings.
